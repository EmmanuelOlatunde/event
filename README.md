# 🎉 Event Booking System for Wedding Planners

A modern, conversion-focused booking system designed to replace messy Instagram/WhatsApp inquiries with a structured, professional workflow.

This project helps event planners manage client requests, packages, and bookings in one simple system.

## 📌 Problem It Solves

Most event planners currently rely on:
- Instagram DMs  
- WhatsApp messages  
- Manual tracking (notebooks, notes apps, memory)

This leads to:
- Lost client requests  
- Delayed responses  
- Poor organization  
- Missed bookings  

This system replaces that chaos with a **structured booking workflow**.

---

## 💡 Solution

A clean web-based booking system that allows clients to:

- View services and packages
- Submit structured booking requests
- Receive confirmation after submission

And allows admins to:

- View all bookings in one dashboard
- Track booking status (New, Contacted, Confirmed)
- Manage client requests efficiently

---

## 🧱 Features

### 🏠 Landing Page
- Hero section with clear value proposition
- Services overview
- "How it works" explanation
- Testimonials section
- Strong call-to-action

### 📦 Packages Page
- 3 structured pricing tiers:
  - Basic
  - Standard (highlighted)
  - Premium
- Feature comparison
- “Book this package” CTA

### 📝 Booking Form
- Clean, structured client intake form
- Fields include:
  - Name
  - Email
  - Phone
  - Event type
  - Event date
  - Budget range
  - Selected package
  - Additional notes

### ✅ Booking Confirmation
- Confirmation message after submission
- Summary of booking request
- Next steps for client

### 📊 Admin Dashboard
- View all booking requests
- Status tracking:
  - New
  - Contacted
  - Confirmed
- Simple and clean management interface
